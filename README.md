## Yeet Sort 



A sorting algorithm that yeets big numbers to the end that is similar to bubble sort.



## How it Works

1. It looks for a conflict in which a big number is in infront of a small one and then swaps them.

3. But if the number is still sitting in front of a small one, instead of swapping, it swaps to the index after the next, basically yeeting it to the end or until the condition matches instead of swapping again and again.
